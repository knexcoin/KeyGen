# KnexCoin Key Generator

Welcome to the KnexCoin Key Generator wiki.

## Overview

`knex-keygen` is a secure offline tool for generating KnexCoin keypairs. It supports both classical **Ed25519** and hybrid **Ed25519 + ML-DSA-65 (FIPS 204)** key generation with post-quantum protection.
